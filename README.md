# Centro de Soporte FT

Sistema interno de gestión de reportes técnicos para las sucursales de FT.

## ¿Para qué sirve?

Permite que las sucursales reporten fallas o problemas técnicos directamente al equipo de Sistemas, sin necesidad de llamadas ni mensajes. Cada reporte queda registrado, tiene seguimiento y el equipo de Sistemas puede gestionarlo desde un panel centralizado.

## ¿Para quién es?

- **Sucursales** — para levantar reportes de fallas (equipo, PVWIN, cancelaciones de documentos, etc.) y consultar el estado de sus reportes anteriores sin necesidad de usuario ni contraseña, solo seleccionando su sucursal.

- **Equipo de Sistemas** — para ver todos los reportes en tiempo real, cambiar su estado, marcarlos como urgentes, agregar comentarios internos, consultar estadísticas y exportar información a Excel.
